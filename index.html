Coding Dojo Algorithm Challenges

Taking the algorithms from Coding Dojo's pre-bootcamp courswork and turning them into functional examples that users can play with and learn from. All functions are made without any pre-built JavaScript functions or methods except .push() and .pop(). View on GitHub.

1. Print 1 to 5,000

Description: This program can print all the numbers in consecutive order from 1 to 5,000.



To make this algorithm, we can use a for loop. We set the initial value of i equal to 1, and stop the loop when i<=5000. We increment i up by 1 with the syntax i++ (i.e. i = i+1).

for (i=1; i<=5000; i++) {
   console.log(i);
}

2. Print Odd Numbers

Description: This program can print all the odd numbers from 1 to 5,000.



To make this algorithm, we can use a for loop like we did in the first example. We set the initial value of i equal to 1 again, but stop the loop when i<=1000. We then increment i up by 2 with the syntax i+=2 (i.e. i = i+2).

for (i=1; i<=1000; i+=2) {
   console.log(i);
}

3. Print Sum of ODD Numbers

Description: This program can print the sum of all the odd numbers from 1 to 5,000,000



To make this algorithm, we can use a for loop like we did in the first two examples. However, this time we need to introduce a variable, sum, and set it equal to 0. Then we set the initial value of i equal to 1 again, but stop the loop when i<=5000. We can increment i up by 2 with the syntax i+=2 (i.e. i = i+2). In order to get the sum of each iteration, we need to use the logic, "sum = sum + i" (or sum += i).

for (i=1; i<=5000; i+=2) {
    sum = sum + i;
}
console.log(sum);

4. Find Max & Min

Description: Given an array with multiple values (e.g. [-3, 3, 5, 7]), this program will find and print the largest and smallest numbers in the array. Toggle the drop-down menus below to set the size of the array (i.e. how many numbers it contains) and the range for how big your numbers can be. This algorithm will find the "max" and "min" values in your array.



The algorithm to find max and min looks like this:

x = [1,2,3,4,5];
var max = x[0];
var min = x[0];

for (i=1; i < x.length; i++) {
    if (x[i] > max) {
    max = x[i];
   }
    if (x[i] < min) {
    min = x[i];
   }
}
console.log(max);
console.log(min)

As you can see, we set two reference values, max and min, equal to x[0], or the first value of the array. As we iterate through the array, each time we compare max and min to the current array value (i.e. x[i]). If x[i] greater than "max", we update "max" so that "max" is equal to x[i]. In other words, each time x[i] is greater than the variable "max", the variable "max", will be updated. The same is true for the variable "min", except "max" is only updated if x[i] is less than "min". In this way, we can find the largest and smallest values in the array.

5. Find Average of Array

Description: Given an array with multiple values (e.g. [1,3,5,7,20]), this program will calculate the average of the values in the array. Toggle the drop-down menus below to set the size of the array (i.e. how many numbers it contains) and the range for how big your numbers can be.



To complete this algorithm, we need to first calculate the sum of the array. We do this by creating a variable, sum = 0. Then we iterate through our array and update the value of sum with the logic, sum = sum + x[i]. After this, we create a variable, average, which equals sum/x.length, and then print the value of "average" to the console.

x = [1,2,3,4,5];
sum = 0;
for (i=0;i < x.length; i++) {
    sum = sum + x[i];
}
var average = sum / x.length;
console.log(average);

6. Square the Values of an Array

Description: Given an array x (e.g. [1, 5, 10, -2]), this algorithm will square each value in the array. When the program is done, x should have values that have been squared (e.g. [1, 25, 100, 4]). Toggle the drop-down menus below to set the size of the array (i.e. how many numbers it contains) and the range for how big your numbers can be. Then sit back and watch the numbers in your array get squared!



To complete this algorithm, we create a for loop that will iterate through each value of the array. Upon each iteration we update x[i] so that x[i] = x[i]*x[i]. Essentially, we are accomplishing our objective by multiplying each value in the array by itself.

x = [2,4,6,8];
for (i=0; i < x.length; i++) {
    x[i] = x[i]*x[i];
}
console.log(x);

7. Eliminate Negative Numbers

Description: Given an array x (e.g. [1,5, 10, -2]), this algorithm replaces any negative number in the array with the value of "0" (zero). When the program is done, x should have no negative values (e.g. [1, 5, 10, 0]). Toggle the drop-down menus below to set the size of the array (i.e. how many numbers it contains) and the range for how big your numbers can be. Those negative numbers will be transformed into zeros in a blink of an eye.



To create this program, we need a conditional statement which can evaluate if a number in the array is less than 0 (i.e. a negative number). So we use a for loop to iterate through the array. Each time we check if x[i] < 0 -- in which case we update x[i] to be equal to 0.

x = [1, 2, 3, 4, -5];
for (i=0; i < x.length; i++) {
    if (x[i] < 0) {
      x[i] = 0;
    }
}
console.log(x);

9. Greater than Y

Description: This program takes an array and returns the number of values in that array whose value is greater than y. For example, if array x = [1, 3, 5, 7] and y = 3, after this program is run it will print 2 (since there are two values in the array whose value is greater than 3). Set a value for Y, then toggle the drop-down menus below to set the size of the array (i.e. how many numbers it contains) and the range for how big your numbers can be.



To make this program, we need two reference variables, "y" and "count". The variable "y" is set to whatever value you choose, for example, 2. "count" is set to zero. We then use a for loop to iterate through our array, and increment count up by one each time x[i] > y. After the program is finished running, count will be equal to the number of values in our array that are greater than y.

x = [1, 2, 3, 5, -5];
var y = 2;
var count = 0;

for (i=0; i < x.length; i++) {
    if (x[i] > y) {
      count++;
    }
}
console.log(count);

10. Bubble Sort

Description: Given an array with multiple random numbers, write a program that will sort the numbers (with the smallest number on the left and the largest number on the right), using the bubble sort method. Toggle the drop-down menus below to set the size of the array (i.e. how many numbers it contains) and the range for how big your numbers will be. Watch as bubble sort works it's magic!



The format of a typical bubble sort is as follows:

x = [0, 5,3,4,-1,-2,5,6];

for (i=0; i < x.length-1; i++) {
   for (j=0; j < x.length-1; j++) {
       if (x[j] < x[j+1]) {
         temp = x[j];
         x[j] = x[j+1];
         x[j+1] = temp;
       }
   }
}
console.log(x);
